## Status Update Protocol

When completing a chapter rewrite:

1. Update the chapter row in the table above:
   - Change Status to `COMPLETED`
   - Record final Word Count
   - Set Started and Completed dates
   - Add any relevant Notes
2. Update the Summary counters
3. Update the Act Progress table
4. Add a log entry to `writing-artifacts/logs/rewrite-pass-log.md`
5. Unblock the next chapter by updating its Notes field
