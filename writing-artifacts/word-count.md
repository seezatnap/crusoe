Update Procedure:

1. Count words with `wc -w`.
2. Update `words`, `delta_to_min`, and `status`.
3. Set `updated_utc` using ISO-8601.
4. Keep row for each chapter until status is `archived` after release snapshot.

Status definitions:

- `draft`: file exists but chapter draft has not started.
- `in_progress`: drafting in progress.
- `blocked`: missing dependency artifacts prevent forward progress.
- `ready_for_review`: chapter draft complete and ready for review.
- `approved`: chapter meets target range and continuity checks.
- `revisions_needed`: chapter needs updates after review or continuity pass.
- `archived`: released or historical snapshot row.
